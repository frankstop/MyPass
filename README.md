# MyPass

MyPass turns normal interests into investor-grade personality debt.

Static GitHub Pages satire app. No build step. No backend. No mercy.

## Features

- Passion profile generator
- Authenticity optimizer with escalating copy rot
- ComedyBot-style joke variants
- Portfolio launderer for tiny projects
- Drag/drop Top 8 Passions board
- Investor deck generator
- Guestbook with localStorage praise laundering
- Shareable profile state through URL hash
- Reduced motion toggle

## Run

Open `index.html`, or serve folder:

```bash
python3 -m http.server 8080
```

## Deploy

Enable GitHub Pages from `main` branch root.
